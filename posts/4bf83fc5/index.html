安装之前要把电脑上原有的node卸载教程
安装教程

  1. nvm下载选择第三个 nvm-noinstall.zip
  2. 成功安装的nvm标志（命令行：nvm）
  3. 配置淘宝镜像
    1. 打开nvm的安装路径，默认安装路径是：C:\Users\Administrator\AppData\Roaming\nvm
    2. 打开settings.txt，加入下面文件
      root: C:\Users\zhong\AppData\Roaming\nvm  
path: C:\Program Files\nodejs 
arch: 64 
proxy: none
node_mirror: http://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/
    3. nvm安装node
      1. 安装命令(version就是要安装的node版本，官网可以查看)
        nvm install version
    4. 切换node版本
      1. 查询所有的已安装得node版本
        nvmlist
      2. 切换node命令
        nvm use version

      3. 配置npm源

        $ npm config set registry https://registry.npm.taobao.org
// 通过下面的方法验证是否成功
npm config get registry 
// 或
npm info express
